# Simple Interest Calculator

A simple calculator that calculates the interest earned on a principal amount using the simple interest formula.

## Formula

```
Simple Interest = (Principal × Rate × Time) / 100
```

## Features

* Calculate simple interest
* Easy-to-use inputs
* Provides the total amount after interest

## Usage

Enter:

* Principal amount
* Interest rate
* Time period

The calculator will display the simple interest and total amount.
